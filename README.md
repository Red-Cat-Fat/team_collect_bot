# Telegram Poll Bot

Бот для группового чата с темами (forum topics), который опрашивает участников о посещении игры.

## Функционал

- Админ устанавливает дату игры командой `/set_game_date DD-MM`
- За день до игры бот автоматически отправляет опрос в заданный топик с кнопками «Да»/«Нет»
- Участники голосуют нажатием кнопок
- Бот обновляет сообщение опроса, показывая текущие голоса
- За день до игры бот публикует список едущих
- Команда `/who_is_going` — мгновенный вывод списка едущих (только для админов)

## Команды

| Команда | Описание | Права |
|---------|----------|-------|
| `/set_game_date DD-MM` | Установить дату игры (год — текущий или следующий, если дата прошла) | Админ |
| `/who_is_going` | Показать список тех, кто нажал «Да» | Админ |

## Быстрый запуск

1. Скопируйте `.env.example` в `.env` и заполните:
   ```env
   BOT_TOKEN=your_bot_token
   ADMIN_IDS=123456789,987654321
   TOPIC_ID=123
   DB_PATH=./data/bot.db
   TIMEZONE=Europe/Moscow
   ```

2. Узнайте `TOPIC_ID`:
   - Добавьте бота в группу
   - Включите темы (forum topics)
   - Перешлите любое сообщение из нужного топика боту @userinfobot или используйте `message_thread_id` из обновлений

3. Запустите:
   ```bash
   docker compose up -d --build
   ```

## Логика дат

- Формат: `DD-MM` (например, `15-06`)
- Год берется текущий
- Если дата уже прошла в текущем году — используется следующий год
- Опрос отправляется автоматически за 1 день до игры в 10:00 (по таймзоне)
- Напоминание со списком едущих — в тот же день

## Структура БД (SQLite)

- `polls` — опросы
- `votes` — голоса пользователей
- `game_schedule` — расписание игр

## Требования

- Docker и Docker Compose
- Бот должен быть админом в группе (для отправки сообщений в топик)
